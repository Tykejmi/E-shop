# Fake E-shop

![page-header](imgs/githubImg.png)

## Description 
This project represents front-end part for E-shop of fake company "Rent a pet Co."

## Features
- Mobile first design
- Fully responsive
- Image transition on hover
- Drop downs
- Modals
- Etc...

## Try it out: 
https://app.netlify.com/sites/tykejmi-eshop/overview

## Technologies used:
- HTML5
- CSS3
- JS
